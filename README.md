<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

You just need Docker to use this project

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/darakt/eventify.git
   ```
2. Go to your new folder
   ```sh
   cd eventify
   ```
3. Let docker do the rest
   ```sh
   docker-compose up
   ```
